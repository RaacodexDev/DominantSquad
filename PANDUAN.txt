  1. CARA GANTI FOTO PROFIL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Cari baris ini di script.js:
    photo: "https://i.pravatar.cc/300?img=47",

  Ganti link-nya dengan URL foto kamu.

  ── Cara mendapatkan URL foto ──

  Cara 1 — Imgur (paling mudah):
    a. Buka https://imgur.com
    b. Klik "New post" → upload foto kamu
    c. Setelah upload, klik kanan foto → "Copy image address"
    d. Paste link tersebut ke bagian photo

  Cara 2 — Google Drive:
    a. Upload foto ke Google Drive
    b. Klik kanan foto → "Get link"
    c. Ubah akses menjadi "Anyone with the link" → klik Copy
    d. Link asli: https://drive.google.com/file/d/ABCD1234/view
    e. Ubah jadi: https://drive.google.com/uc?id=ABCD1234
       (salin ID-nya saja, ganti "ABCD1234" dengan ID fotomu)

  Contoh hasil:
    photo: "https://i.imgur.com/Xy8AbC.jpg",

  ⚠️  Tips foto profil terbaik:
     - Ukuran: 400×400 piksel (persegi)
     - Format: JPG atau PNG
     - Pastikan wajah terlihat jelas dan center

  Cari:
    whatsapp: {
      url: "https://wa.me/628123456789",
      sub: "Chat & order langsung 💬",
    },

  Format URL WhatsApp: https://wa.me/62 + nomor (tanpa 0 di depan)

  Cara konversi nomor:
    Nomor HP  : 0812-3456-7890
    Hilangkan 0: 812-3456-7890
    Tambah +62 : 62812-3456-7890
    Tanpa strip: 6281234567890
    Hasil URL  : https://wa.me/6281234567890

  ⚠️  Tidak boleh ada spasi, tanda hubung (-), atau tanda plus (+)

  Contoh:
    url: "https://wa.me/6281234567890",
    sub: "Mau order? Hubungi aku! 🛒",

  CARA UPLOAD SUPAYA BISA DIPAKAI DI BIO IG / TIKTOK (GRATIS)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Opsi 1 — Netlify Drop (TERMUDAH, tidak perlu akun):
    1. Buka: https://app.netlify.com/drop
    2. Drag & drop FOLDER yang berisi ketiga file
       (index.html, style.css, script.js)
    3. Tunggu beberapa detik
    4. Dapat link langsung, contoh: https://fancy-cat-123.netlify.app
    5. Copy link itu, paste ke bio Instagram / TikTok!

  Opsi 2 — GitHub Pages (permanen, gratis):
    1. Daftar di https://github.com
    2. Buat repository baru (klik tanda +, pilih New repository)
    3. Upload ketiga file ke repository tersebut
    4. Buka Settings → Pages → pilih branch "main" → Save
    5. Dapat link: https://usernamegithub.github.io/nama-repo

  TIPS & CATATAN PENTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✅ DO (yang harus dilakukan):
     - Simpan file setelah mengedit (Ctrl+S)
     - Pastikan tanda kutip ("") tetap ada di awal dan akhir teks
     - Pastikan tanda koma (,) tetap ada di akhir setiap baris
     - Test tampilan di HP sebelum pasang di bio

  ❌ DON'T (yang tidak boleh dilakukan):
     - Jangan hapus tanda { } [ ] atau , di luar teks
     - Jangan ada nomor WA dengan spasi atau tanda strip
     - Jangan rename file (harus tetap index.html, style.css, script.js)

  💡 TIPS:
     - Buka index.html di browser Chrome/Firefox untuk preview
     - Tekan F12 untuk lihat apakah ada error (di tab Console)
     - Jika tampilan tidak berubah setelah edit, tekan Ctrl+F5
       untuk hard refresh browser
